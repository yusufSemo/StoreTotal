# Store Income Analysis App

The Store Income Analysis App is a JavaFX application designed to help store owners analyze and calculate their daily income. It provides a user-friendly interface for inputting sales data and offers various analysis tools, including custom-created graphs for different payment methods (Cash, Card, Online Order).

## Features

- **Graphical Representation:** Visualize daily income breakdown using custom graphs for Cash, Card, and Online Order totals.

- **Analysis Tools:** Calculate and analyze total income for the day, week, month, etc.

## Prerequisites

Make sure you have the following installed before running the application:

- Java JDK (version 21.X.X)
- JavaFX SDK (version 21.X.X)
